ABOUT THIS PROJECT

Virtual Labs - Pawar Public School Nanded City

Project Overview

This Virtual Labs website was developed as a school project to showcase the laboratories of Pawar Public School Nanded City for students.

The website provides an interactive platform where students can explore different labs, learn about famous scientists in each field, and understand the location and facilities of each laboratory.

Features

  • Interactive lab exploration
  • Information about famous scientists
  • Floor-wise lab locations
  • Wikipedia integration for scientists
  • Modern, responsive design
